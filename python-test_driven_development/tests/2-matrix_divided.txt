Using matrix_divided

Import the function:
>>> matrix_divided = import('2-matrix_divided').matrix_divided

Basic valid usage:
>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> matrix_divided(matrix, 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Verify original matrix is unchanged:
>>> print(matrix)
[[1, 2, 3], [4, 5, 6]]

Negative numbers:

>>> matrix_divided([[10, 20], [30, 40]], -2)
