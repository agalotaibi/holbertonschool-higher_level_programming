==============================
matrix_divided
==============================

This file tests the matrix_divided function.

Correct division:
>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> matrix_divided(matrix, 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Original matrix should not be changed:
>>> matrix
[[1, 2, 3], [4, 5, 6]]

Matrix with floats:
>>> matrix_divided([[1.5, 2.5], [3.5, 4.5]], 2)
[[0.75, 1.25], [1.75, 2.25]]

Negative numbers:
>>> matrix_divided([[-1, -2], [-3, -4]], 2)
[[-0.5, -1.0], [-1.5, -2.0]]

div as float:
>>> matrix_divided([[2, 4], [6, 8]], 2.0)
[[1.0, 2.0], [3.0, 4.0]]
