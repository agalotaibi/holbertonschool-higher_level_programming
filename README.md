# Holberton School - Higher Level Programming

This repository contains the projects and exercises for the **Higher Level Programming** curriculum at Holberton School. The projects cover a wide range of topics including Python programming, Object-Oriented Programming (OOP), SQL database management, Web Development (JavaScript), and API interaction.
