# JavaScript - DOM Manipulation
This project focuses on the Document Object Model (DOM) and how to manipulate it using JavaScript. The tasks cover selecting HTML elements, modifying styles and classes, handling events (clicks), creating new elements, and retrieving data from external APIs using Fetch.

# Concepts Covered
- document.querySelector
- element.style
- element.classList (add, toggle)
- element.textContent
- document.createElement & appendChild
- addEventListener (click events)
- Fetch API & Promises
- DOMContentLoaded event
